Crez une carte choroplthe

Carte de population de Bouira

Une carte choroplthe (du grec : zone/rgion et : grand nombre, multitude ) est une carte thmatique o les rgions sont colores ou remplies d'un motif qui montre une mesure statistique, tels la densit de population ou le revenu par habitant. Ce type de carte facilite la comparaison d'une mesure statistique d'une rgion l'autre ou montre la variabilit de celle-ci pour une rgion donne.wikipedia

Prparation de donnes

Nous allos crer une carte choplthe pour visualiser gographiquement la population de Bouira, et pour cela il faut avoir la frontire Administrative de Bouira et aussi tlcharger le fichier CSV de la Populations de Bouira

Source du fichier popultion: ONS

Cartes choroplthes

GeoPandas facilite la cration de cartes Choropleth (cartes o la couleur de chaque forme est base sur la valeur d'une variable associe). Utilisez simplement la commande plot avec l'argument de colonne dfini sur la colonne dont vous souhaitez utiliser les valeurs pour attribuer des couleurs.

In[9]:
 # Plot by population

merged.plot(column="Total");

Crer une lgende

In[10]:
merged.plot(
    column="Total",
    legend=True,
    legend_kwds={"label": "Population 2008", "orientation": "horizontal"},
);

Toutefois, lapparence par dfaut des axes de lgende et de trac nest peut-tre pas souhaitable. On peut dfinir les axes de trac (avec ax) et les axes de lgende (avec cax), puis les transmettre l'appel plot(). L'exemple suivant utilise mpl_toolkits pour aligner horizontalement les axes du trac et les axes de la lgende et modifier la largeur:

In[11]:
import matplotlib.pyplot as plt

from mpl_toolkits.axes_grid1 import make_axes_locatable

fig, ax = plt.subplots(1, 1)

divider = make_axes_locatable(ax)

cax = divider.append_axes("bottom", size="5%", pad=0.1)

merged.plot(
    column="Total",
    ax=ax,
    legend=True,
    cax=cax,
    legend_kwds={"label": "Population in 2008", "orientation": "horizontal"},
);

Choisir les couleurs

Vous pouvez galement modifier les couleurs utilises par plot() avec l'option cmap. Pour une liste complte des palettes de couleurs, consultez Choisir des palettes de couleurs dans Matplotlib.

In[12]:
merged.plot(column='Total', cmap='OrRd');
