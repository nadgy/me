QuickOSM

  1. Exécutez le plugin à partir du menu Vecteur ► QuickOSM ► QuickOSM….

  2. Dans la zone préreglage, tapez boundary dans le menu déroulant clé.

  3. Laissez le champ Valeur vide.

  4. Sélectionnez dans dans le menu déroulant suivant, et ecrire bouira bouira Algeria

  5. Développez le groupe avancé ci-dessous et décochez tous les types de géométrie sur la droite sauf Multipolygones.

  6. Pressez executer requête

Une nouvelle couche bouira bouira Algeria est ajoutée au panneau couches, montrant les limites administrative de bouira dans l’étendue sélectionnée.

Symbologie

    pour les 6 étapes suivantes sélectionnez Emprise d'une couche dans le menu déroulant suivant, et ecrire le nom de la couche masque bouira pour toutes les autres couches

  1. Procédez comme ci-dessus pour extraire d’autres données :

    1. Key = landuse et type de géométrie Multipolygons.

    2. Key = natural, Value = water et type de géométrie Multipolygons.

    3. Key = highway et vérifiez les types de géométrie Lines et Multilines.

    4. Key = building, et type de géométrie Multipolygons.

    5. Key = waterway, Value = river et vérifier les types de géométrie Lines et Multilines.

    6. Key = place et type de géométrie Points.

Réorganisation des calques

Les calques dans votre liste de calques sont dessinés sur la carte dans un certain ordre. Le calque en bas de la liste est dessiné en premier, et le calque en haut est dessiné en dernier. En changeant leur ordre dans la liste, vous pouvez changer l’ordre suivant lesquel ils sont dessinés.
